# Installation

python-gazelle is published on PyPI and requires **Python 3.11+**.

=== "uv"

    ```bash
    uv add python-gazelle
    ```

=== "pip"

    ```bash
    pip install python-gazelle
    ```

That's all you need to start making API calls. Continue to the
[Quick start](quickstart.md).

!!! note "Installing for development"
    If you want to work on python-gazelle itself (not just use it), see the
    [Development](../development.md) page for the uv-based contributor setup.
